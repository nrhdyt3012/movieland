🎬 Movieland

Movieland adalah sebuah platform streaming berbasis web yang dikembangkan menggunakan PHP Native dan MySQL (XAMPP). Proyek ini bertujuan untuk memberikan pengalaman menonton film atau video yang menyenangkan, praktis, dan berkualitas tinggi langsung dari browser pengguna, tanpa perlu menginstal aplikasi tambahan.

📌 Latar Belakang

Dalam era digital saat ini, layanan streaming menjadi bagian penting dalam industri hiburan. Dengan meningkatnya akses internet dan perangkat pintar, masyarakat cenderung lebih memilih menonton konten secara online dibandingkan metode konvensional seperti DVD atau TV kabel.

Movieland hadir sebagai solusi inovatif yang menggabungkan kemudahan akses, antarmuka pengguna yang intuitif, serta berbagai fitur penunjang pengalaman menonton yang lebih baik.

❓ Rumusan Masalah

Beberapa pertanyaan utama dalam pengembangan Movieland antara lain:

- Bagaimana merancang antarmuka pengguna (UI) yang intuitif dan responsif?
- Fitur apa saja yang perlu disediakan untuk memenuhi kebutuhan pengguna dalam menonton video secara online?
- Bagaimana cara mengelola konten multimedia dalam skala besar dengan performa yang stabil?
- Bagaimana sistem keamanan dan manajemen pengguna diterapkan?
- Bagaimana mengadopsi teknologi web terbaru (HTML5, CSS3, JavaScript) agar kompatibel di berbagai perangkat?

📦 Fitur Utama

- 📺 Pemutaran Video: Menonton film/video.
- 🔍 Pencarian Konten: Fitur search yang memudahkan pengguna menemukan film yang diinginkan.
- 📁 Kategori & Daftar Putar: Konten dikelompokkan dalam kategori serta mendukung fitur playlist.
- 👤 Manajemen Pengguna: Sistem login dan registrasi untuk mengatur preferensi pengguna.

 🚫 Batasan Masalah

- Fokus pada aspek teknis pengembangan, bukan produksi atau lisensi konten.
- Penggunaan teknologi web standar (HTML5, CSS3, JS, PHP Native).
- Tidak membahas aspek monetisasi atau strategi bisnis.
- Aplikasi berbasis web, tidak dikembangkan untuk Android/iOS native.
- Pembahasan keamanan dibatasi pada aspek dasar.

 🎯 Tujuan Proyek

1. Merancang UI yang user-friendly dan responsif.
2. Mengembangkan fitur utama seperti pemutar video, pencarian, playlist, dan rekomendasi.
3. Menyediakan layanan streaming yang stabil.
4. Menyediakan sistem keamanan dan manajemen pengguna dasar.
5. Menggunakan teknologi web modern agar kompatibel di banyak perangkat.

 🛠️ Teknologi yang Digunakan

- PHP Native (Tanpa framework)
- MySQL via XAMPP
- HTML5, CSS3, dan JavaScript
- Bootstrap (untuk desain responsif)

 🧪 Cara Menjalankan

1. Clone repositori ini ke folder `htdocs` milik XAMPP:

   ```bash
   git clone https://github.com/nrhdyt3012/movieland.git
````

2. **Aktifkan Apache dan MySQL** melalui XAMPP Control Panel.

3. **Import database**:

   * Buka `phpMyAdmin`.
   * Buat database baru (misalnya `movieland`).
   * Import file SQL yang disediakan di folder proyek (misalnya `movieland.sql`).

4. **Akses aplikasi**:
   Buka browser dan kunjungi:

   ```
   http://localhost/movieland-main/movieland
   ```
